Node Deletion Test Instructions

  1. Open the app at http://localhost:5176/
  2. Click on the canvas to add nodes (should create Rect1, Rect2, etc.)
  3. Click on a node to select it (should show selection outline)
  4. Press Delete or Backspace key
  5. The selected node should be deleted

Expected Results:

  • ✅ Node gets selected when clicked (visual feedback)
  • ✅ Delete/Backspace key removes the selected node
  • ✅ Multiple nodes can be selected with Shift+Click
